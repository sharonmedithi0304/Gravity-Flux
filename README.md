# 🚀 Gravity Flux

An interactive physics playground built using HTML, CSS, and JavaScript.  
Simulates gravity, collisions, and real-time user interaction in a dynamic environment.

## ✨ Features

- Real-time gravity simulation  
- Drag and throw objects  
- Object-to-object collision system  
- Explosion effect on click  
- Dynamic object spawning  
- Gravity toggle (ON/OFF)  
- Smooth animations and rotations  
- Interactive UI controls  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## 🎮 How to Use

- Click anywhere → spawn objects + trigger explosion  
- Drag objects → move them freely  
- Release → throw with momentum  
- Toggle Gravity → enable/disable gravity  
- Clear → remove all objects
